# New! 〇✕ゲーム

伝統的な〇✕ゲームに、「3つ並ぶと消滅する罠」と「盤面全体を傾ける重力」という2つの特殊ルールを融合させた、新感覚の戦略パズル対戦ゲームです。

初心者向けの親切なガイド機能から、2手先を読んでプレイヤーの罠を回避する最高難易度のAIまで、幅広いプレイヤーが楽しめるように設計されています。

---

## 🎮 ゲームルール

ベースは **6×6の盤面** で行う〇✕ゲームですが、以下の特殊ルールが存在します。

### 1. 🏆 勝利条件
* 縦、横、斜めのいずれかに、自分のコマを **4つ以上** 連続して並べたプレイヤーが勝利となります。

### 2. 👻 3つ並び消滅の罠（絶対の悪）
* 縦、横、斜めのいずれかに、自分のコマが **「ちょうど3つ」** 並んでしまうと、その3つのコマは盤面から **即座に消滅** します。
* 4つ並べるための布石を打つ際、うっかり3つ並びを作ってしまうと自爆してしまうため、綿密な計算が必要です。

### 3. 🌎 1回限りの「重力発動」
* 各プレイヤーは、1ゲーム中に **1度だけ** 「重力」ボタンを使用できます。
* 重力を使用すると、盤面上の全てのコマが指定した方向（↑・↓・←・→）に隙間なく引き寄せられます。
* コマが移動した結果、新しく「ちょうど3つ」並んだ場合は **連鎖して消滅** し、空いた隙間を埋めるようにさらにコマが落下します（連鎖重力システム）。

---

## 🤖 CPU対戦モード（3つの難易度）

プレイヤーのレベルに合わせて、思考ロジックの異なる3つのAIを搭載しています。

* 🟢 **イージー (Easy)**
  * 重力を一切使用しないモードです。1手詰みの勝利や基本的なリーチ阻止は行いますが、特殊ルールに慣れたい初心者向けです。
* 🟡 **ノーマル (Normal)**
  * 重力での一発勝利や、重力を使った高度なリーチ防御（自爆を避けるための盤面シャッフル）を使いこなす直感型のAIです。
* 🔴 **ハード (Hard) 【最凶】**
  * **2手先（プレイヤーの次のカウンター）を頭の中でシミュレーションする超戦略型AI** です。プレイヤーが仕掛ける「次のターンで重力でハメる」といったトラップを事前に察知し、徹底的に回避してきます。勝率は五分五分、上級者向けの最高難易度です。

## ✨ 充実の機能

* 📋 **ビジュアルヘルプ**: ルール画面には、手書きの複雑な盤面から「下重力 ➔ 3つ消滅 ➔ 空きを埋める再落下」までの連鎖プロセスがひと目でわかる4×4の精密な図解を搭載。
* ⚙️ **初心者ガイドモード (ON/OFF選択制)**:
  * **自爆警告**: マウスを空きマスに乗せた際、そこが「3つ並んで消滅する危険マス」だった場合、セルが赤く点滅して警告します。
  * **重力点滅プレビュー**: 重力方向ボタンにマウスを乗せると、盤面全体が「現在の状態」と「重力移動後の未来の姿」で1秒間隔で交互に点滅し、移動ルートを完璧に予測できます。
* ⚡ **ゲームテンポ調整**: 「じっくり（アニメーションあり）」と「サクサク（超高速処理）」をいつでも切り替え可能。
* 🌙 **ダークモード搭載**: 目に優しいダークテーマに1クリックで切り替えられます。
